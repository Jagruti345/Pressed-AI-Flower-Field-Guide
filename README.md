## ✨ Features

🌼 AI-powered flower recognition

📷 Image upload with preview

⚡ Fast prediction using AWS Lambda

🌐 REST API powered by Amazon API Gateway

📱 Fully responsive design

☁️ Serverless backend

🚀 Frontend hosted on Amazon EC2

🔄 Automatic deployment using GitHub Actions

🎨 Modern user interface

```bash
git clone https://github.com/Jagruti345/Pressed-AI-Flower-Field-Guide.git
```

```bash

### Run

```

in your browser.
